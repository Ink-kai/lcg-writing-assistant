# 人机分工

## 你的角色（人工）

- **插件开发**：用 Claude Code 迭代开发
- **社区运营**：在 Obsidian 论坛/Reddit 发帖互动
- **用户支持**：回复 GitHub Issues

## Claude Code 的角色（AI）

- **代码开发**：编写、调试、重构插件代码
- **文档撰写**：写 README、内容草稿
- **研究支持**：调研竞品、平台规则

---

## 分工原则

| 类型 | 谁做 | 原因 |
|------|------|------|
| 代码开发 | Claude Code + 你 | 你是开发者，AI 是辅助 |
| 社区互动 | 你 | 需要个人化，不能 AI 冒名 |
| 内容创作 | AI 起草 + 你修改 | AI 快速生成初稿，你把关质量 |
| 用户反馈分析 | 你 + AI | 你理解用户，AI 帮你整理 |

---

## 当前阶段

MVP 验证阶段，核心动作是：
1. 等插件上线
2. 发内容试水
3. 观察结果

Claude Code 可以帮你准备内容草稿，但不能代替你发帖。
